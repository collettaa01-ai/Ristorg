# Ristorg Design System

## Color Strategy
Restrained: tinted neutrals + indigo accent. Expanding to include semantic colors.

## Current Tokens
```css
--bg: #f5f6fa;
--surface: #ffffff;
--text: #1a1a2e;
--text-secondary: #6b7194;
--accent: #4f46e5;
--accent-light: #eef2ff;
--accent-hover: #4338ca;
--border: #e5e7ef;
```

## Typography
- Font: Segoe UI / system stack
- Body: 15px base
- Heading scale: 1.5rem (area title), 1.15rem (topbar), 1rem (card names)
- Secondary text: .82rem-.95rem range (needs consolidation)

## Spacing
- Page padding: 32px (20px mobile)
- Card padding: 28px 24px
- Row padding: 14px 18px
- Gap: 20px (cards), 10px (rows)

## Border Radius
- Large: 14px (cards, modals)
- Medium: 10px (inputs, buttons, rows)
- Small: 8px (icon buttons)

## Elevation
- sm: 0 1px 3px rgba(0,0,0,.06)
- md: 0 4px 16px rgba(0,0,0,.08)
- lg: 0 8px 32px rgba(0,0,0,.12)

## Components
- Area cards (icon + title + description)
- Operator rows (info + action buttons)
- Shift cards (header badge + table + footer actions)
- Sub-tabs (underline active indicator)
- View selector (segmented control)
- Date navigation (arrows + label)
- Modals (header + body + footer)
- Sync indicator (dot + label badge)
